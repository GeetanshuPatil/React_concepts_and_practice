# React Concepts and Practice 🚀

This repository contains my structured learning of React, organized topic-wise with hands-on examples and mini implementations.

## 📚 Topics Covered

- JSX
- Props
- State Management
- Event Handling
- Form Handling
- Routing & Nested Routing
- Redux (Basics)
- useReducer
- useCallback
- useMemo
- React.memo
- Component Structure (Navbar, Aside, etc.)

## 🎯 Purpose

The goal of this repository is to:
- Build a strong foundation in React
- Understand core concepts through practical examples
- Prepare for real-world frontend development

## 📁 Structure

Each folder represents a specific React concept with its own implementation and examples.
